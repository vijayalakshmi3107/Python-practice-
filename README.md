# Python Basics Programs

This repository contains beginner-level Python programs created while learning core concepts like conditions, loops, lists, and functions.

## Programs Included

- Hello World Program
- Multiplication Table (while loop)
- Multiplication Table (for loop)
- Temperature Checker (Hot/Warm/Cool/Cold)
- Even/Odd Checker
- Positive/Negative/Zero Checker
- Count Numbers Divisible by 3 and 5
- Cube of Numbers using List
- Basic Calculator (if-else)
- Calculator using Functions
- Addition using Function
- Factorial Program (Function)

## Concepts Covered

- Input/Output
- Conditional Statements (if, elif, else)
- Loops (for, while)
- Functions
- Lists
- Basic Problem Solving

## Language Used

Python
